# AviNation (TR-only)

## Local run
```bash
npm install
npm run dev
```

## Deploy (Netlify - Git)
- Build command: `npm run build`
- Publish directory: `.next`
- Node version: `20`
- Netlify Next.js plugin is configured via `netlify.toml`.

## Deploy (Vercel)
Import the GitHub repo in Vercel and deploy (no extra config needed).
